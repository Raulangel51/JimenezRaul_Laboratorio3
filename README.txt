Este laboratorio servirá como una introducción para el alumno al sistema de UI de Unity. El alumno puede usar su Laboratorio 2 (preferiblemente) como base para hacer este laboratorio, agregándole las interfaces necesarias. De igual manera, el alumno puede elegir crear otro juego base diferente para el propósito de este lab. Para recibir la nota total del laboratorio, el estudiante debe agregar a su Lab 2 (u otro juego base que se haya escogido) como mínimo lo siguiente:

Un menú principal (una escena separada de la del "primer nivel" o escena del juego) con algún título para el lab/juego, con por lo menos un botón "Iniciar" que cambie a la escena con el juego. El estudiante puede agregar los componentes que desee para llegar a un diseño del menú principal que le guste.
Un menú de pausa en la escena del juego. Al presionar ESC, el menú de pausa debe aparecer y la escena se debe pausar (que los objetos se dejen de "updatear" y que otros botones en la escena no se puedan presionar). Al presionar ESC otra vez, se sale de la pausa. De igual manera, el menú de pausa debe incluir un botón "Continuar" que quite la pausa y otro "Salir al Menú Principal" que cambie a la escena de menú principal.
Un botón en la escena del juego que permita que se reinicie el nivel desde el principio.
Mostrar cierta información del juego en el UI, ya sea usando imágenes o texto. Por ejemplo, para los alumnos que implementaron recolección de monedas en su Lab 2, mostrar en el UI la cantidad de monedas que se han recogido.
La implementación de cada uno de estos puntos cuenta como un punto de la nota del laboratorio, para un total de 4 puntos. El alumno es libre de usar los modelos, texturas o materiales que quiera, y también se les anima a que experimenten con diferentes diseños de UI e implementando otros componentes y sus funcionalidades. En la página de inicio del curso, en el módulo de la tercera semana, se puede encontrar un archivo comprimido que se puede descargar. Este incluye el archivo ejecutable de un ejemplo para este laboratorio.
